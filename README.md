# Yelbolt — Claude Code Marketplace

Official [Claude Code](https://claude.ai/code) plugin marketplace by [Yelbolt](https://yelbolt.co).

## Add the marketplace

```shell
/plugin marketplace add yelbolt/claude-marketplace
```

## Available plugins

### UI Color Palette

Design color palettes, audit contrast, generate code, and sync to Figma, Penpot, Sketch, and Framer — powered by the [UI Color Palette API](https://ui-color-palette.com).

```shell
/plugin install ui-color-palette@yelbolt
```

**Documentation**: [docs.ui-color-palette.com/claude](https://docs.ui-color-palette.com/claude)

## Update

```shell
/plugin marketplace update yelbolt
```
